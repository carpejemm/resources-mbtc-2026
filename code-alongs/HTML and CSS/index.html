This is a header 1

We should only have one header 1 per page. It is the most important header and should be used for the main title of the page. It is recommended to use only one header 1 per page to maintain a clear hierarchy and structure in the content. Using multiple header 1 tags can confuse search engines and users about the main topic of the page. Instead, you can use header 2, header 3, etc. for subheadings and sections within the page.

This is an image tag. It is used to embed an image in the webpage. The src attribute specifies the URL of the image to be displayed. The alt attribute provides alternative text for the image if it cannot be displayed. It is important to include the alt attribute for accessibility reasons, as it allows screen readers to describe the image to visually impaired users. If the src attribute is empty, no image will be displayed, and the alt text will be shown instead. For example: Image tag is a self-closing tag, which means it does not require a closing tag. Why use Semantic Tags?

Semantic tags help search engines (SEO) and other technologies understand the structure and meaning of the content on a webpage. They provide context to the content, making it easier for users and machines to navigate and interpret the information.
